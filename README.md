### Socho Kam, AI Sambhale Income

FinCARE is an AI-powered personal finance app built on FINOS
(Financial Intelligence & Navigation Operating System).

## What it does
- Locks money for bills & EMIs
- Calculates safe-to-spend balance
- Helps users achieve goals
- Uses AI to reduce financial risk

## Tech Stack
- Frontend: Stitch (UI Builder) + HTML / CSS / JavaScript
- Backend: Firebase (Auth, Firestore)
- AI: Google Gemini API
